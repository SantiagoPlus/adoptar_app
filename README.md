# Adopta App

Adopta App es una aplicación web para gestionar adopciones responsables de animales. Permite publicar mascotas, gestionar solicitudes, administrar perfiles y mantener un seguimiento sanitario en una plataforma centralizada.

## Inicialización Rápida
1. Asegurarse de tener Node.js listo.
2. Configurar el archivo `.env.local` con las variables de Supabase.
3. Instalar dependencias con `npm install`.
4. Iniciar el servidor local con `npm run dev`.
